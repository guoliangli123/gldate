# GlDate类

对原生date的扩展，包括原生Date对象的所用方法，但是并不是真正的Date类型，可通过originalDate属性拿到原生Date。

## Property

- `originalDate` 原始date对象
- `year` get or set
- `month` get or set
- `day` get or set
- `hour` get or set
- `minute` get or set
- `second` get or set
- `timeStamp` get or set

## Method

- `format`
- `getTimeStamp`
- `get12Ts`
- `get12MilliSecondTs`
- `getDateByDayDistance`
